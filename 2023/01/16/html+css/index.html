HTML5+CSS3

Made by Ziyang CHEN, Rocky, 陈子阳

网页最主要由3部分组成：结构、表现和行为。网页现在新的标准是W3C，目前模式是HTML、CSS和JavaScript。

  • HTML (HyperText Markup Language): 构建网页的基础,它通过标签来定义网页的内容和结构。HTML 用来规定网页中的文本,图片,视频等内容,以及它们的语义和结构。
  • CSS (Cascading Style Sheets): 控制网页样式和布局的语言。CSS 用于设置网页的外观样式,如文字颜色、大小、边距、背景等。CSS 可以将样式信息与网页内容分离,使得同一份内容可以使用不同的样式显示。
  • JavaScript: 一种脚本语言，主要用于实现网页的各种交互和动态效果。JavaScript 可以响应用户的交互,控制网页内容,使网页有丰富的动态功能。比如实现网页验证、轮播图、下拉菜单等等。
